# 🔩 Análisis de Precios Unitarios (APU)

> El APU es la **columna vertebral** de la propuesta económica. Su correcta integración es obligatoria por la LOPSRM. Un error aritmético que altere el importe total del catálogo **invalida la propuesta completa**.

---

## ¿Qué es el APU?

Es la **disección matemática** del costo de ejecutar una unidad de medida de un concepto de obra (1 m², 1 m³, 1 ml, 1 pieza, etc.). Demuestra que el precio ofertado es real, sustentable y rentable.

---

## Estructura Completa del APU

```
PRECIO UNITARIO (P.U.)
│
├── COSTOS DIRECTOS (CD)
│   ├── Materiales
│   │   └── Cantidad × precio de mercado del insumo
│   ├── Mano de Obra
│   │   └── Cuadrillas × rendimiento × Factor de Salario Real (FASAR)
│   └── Maquinaria y Equipo
│       └── Costo horario × horas utilizadas por unidad
│
├── COSTOS INDIRECTOS (CI) = % sobre CD
│   ├── Indirectos de campo (oficina de obra, vigilancia, herramienta menor)
│   └── Indirectos de administración central (overhead de empresa)
│
├── FINANCIAMIENTO (FI) = % sobre (CD + CI)
│   └── Costo del dinero por el desfase entre erogaciones y cobros
│
└── UTILIDAD (U) = % sobre (CD + CI + FI)
    └── Margen de ganancia del contratista
```

---

## El Factor de Salario Real (FASAR)

El salario pagado al trabajador es mayor al salario base por las prestaciones de ley:

| Componente | % aprox. sobre salario base |
|---|---|
| Salario base | 100% |
| IMSS (cuota patronal) | ~35% |
| INFONAVIT | ~5% |
| INFOTEC / SAR | ~2% |
| Prima vacacional, aguinaldo, etc. | ~15-20% |
| **Factor de Salario Real (FASAR)** | **~1.55 a 1.70** |

> [!NOTE]
> Si el salario base es $350/día, el costo real para la empresa es: $350 × 1.62 = **$567/día**. No incluir el FASAR correctamente es causa de propuestas "temerarias".

## Integración de Costos Indirectos Típica

| Componente | % sobre Costo Directo (rango típico) |
|---|---|
| Indirectos de campo | 5–9% |
| Indirectos de administración central | 6–10% |
| **Total indirectos** | **11–19%** |
| Financiamiento | 1–3% |
| Utilidad | 5–10% |
| **Total sobrecosto** | **17–32%** |

## Ejemplo: APU Concepto 3 — Carpeta Asfáltica (m²)

*(Del caso Morelia — Constructora Michoacana de Infraestructura)*

| Componente | Descripción | Costo/m² |
|---|---|---|
| **Materiales** | Mezcla asfáltica PG 64-22 en planta + emulsión de liga | $198.50 |
| **Mano de obra** | Cabo + 2 oficiales + 4 peones (cuadrilla tipo) | $32.00 |
| **Equipo** | Extendedora Finisher + rodillo vibratorio + rodillo neumático + camión | $49.00 |
| **COSTO DIRECTO** | | **$279.50** |
| Indirectos (15%) | 8% central + 7% campo | $41.93 |
| Financiamiento (1.5%) | | $4.19 |
| Utilidad (7%) | | $19.57 |
| **PRECIO UNITARIO** | | **$345.19 ≈ $345.00/m²** |

## Errores que Invalidan el APU

> [!CAUTION]
> Los siguientes errores son causas directas de descalificación:

1. **Error aritmético** que altere el subtotal o el precio unitario resultante
2. **Inconsistencia** entre el precio en el APU y el precio en el catálogo de conceptos
3. **FASAR no justificado** — no presentar el análisis del factor salarial
4. **Costo horario de maquinaria no sustentado** — se requiere el análisis de costo horario
5. **Indirectos no desglosados** — no se acepta "20% de indirectos" sin apertura
6. **Precio por debajo del costo directo real** — propuesta temeraria; implica insolvencia
