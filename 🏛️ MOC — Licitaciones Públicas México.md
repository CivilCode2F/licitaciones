## ¿Cómo usar este sistema?

Este MOC (Map of Content) es el **punto de entrada** a todo el conocimiento. Cada nota enlazada es **atómica** — aborda un solo concepto en profundidad. Los enlaces `[[dobles corchetes]]` crean el tejido del cerebro digital.

```
MARCO LEGAL → PROCEDIMIENTOS → PROCESO → DOCUMENTOS → GARANTÍAS → RIESGOS → DEFENSA
      ↑                                                                          ↓
      └──────────────── SIMULACIÓN PRÁCTICA (caso Morelia) ────────────────────┘
```
